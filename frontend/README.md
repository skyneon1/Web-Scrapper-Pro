# Web Scraper Frontend

React frontend application for the Web Scraper Pro.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

The app will open at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

The build folder will be created with optimized production files.

## Features

- **Dashboard**: Overview of scraping statistics
- **Scraper**: Create new scraping jobs
- **History**: View and manage all scraping jobs
- **Analytics**: Visual analytics and charts
- **Settings**: Application settings and information

## Tech Stack

- React 18
- Material-UI (MUI) 5
- React Router 6
- Axios for API calls
- Recharts for data visualization
